# Kristijan Gega Portfolio

Modern dark editorial developer portfolio built with React, TypeScript and Vite.

## Overview

This project is my personal developer portfolio, designed to present my work, skills and contact information in a more polished and professional way.

The portfolio follows a typography-first dark editorial direction with strong visual hierarchy, oversized headings, cleaner section flow and a more premium frontend presentation. The goal was to move beyond a simple student portfolio and create something that feels more intentional, modern and production-ready.

## Live Demo

- **Portfolio:** https://kgega-portfolio.vercel.app/

## Tech Stack

- **React**
- **TypeScript**
- **Vite**
- **CSS**
- **Tailwind CSS**
- **Framer Motion**

## Features

- Typography-first dark editorial landing page
- Oversized responsive hero section
- Sticky header navigation
- Mobile hamburger menu
- About, Services, Projects, Stack and Contact sections
- Improved project section with clearer click targets
- Animated infinite stack carousel with framework icons
- Hover interactions for stack items
- Contact section with icon-based links
- Responsive layout for desktop, tablet and mobile
- Smooth transitions and modern UI polish

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/kgega-dev/kgega-portfolio.git
cd kgega-portfolio
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
